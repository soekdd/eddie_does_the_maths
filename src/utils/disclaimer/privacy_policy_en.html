3. Server log files

When you visit this website, the hosting provider and/or web server automatically collects and stores information in so-called server log files. This data may qualify as personal data (in particular the IP address).

Typically, the following data is processed:

  • IP address (possibly shortened/anonymized, if configured accordingly)
  • Date and time of the request
  • Requested page/file (URL/path)
  • HTTP status code
  • Referrer URL (previously visited page)
  • User-Agent (browser/OS)
